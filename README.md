## ğŸš€ Getting Started

### Option 1: Open Directly
Simply double-click on `index.html` to open the portfolio in your default web browser.

### Option 2: Use Live Server (Recommended)
For the best experience with a local development server:

1. Install [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

### Option 3: Python HTTP Server
```bash
# Navigate to the project directory
cd "d:\imp notes"

# Python 3
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```

### Content
Update the following in `index.html`:

1. **Personal Information**: Name, title, description
2. **About Section**: Bio, experience, highlights
3. **Skills**: Add/remove skills and adjust percentages
4. **Portfolio Projects**: Update project details and images
5. **Contact Information**: Email, phone, location
6. **Social Links**: Add your GitHub, LinkedIn, Twitter URLs

### Images
Replace the placeholder images with your own:
- `profile-image.jpg` - Your professional photo
- `about-image.jpg` - Working/casual photo
- `project1-6.jpg` - Screenshots of your actual projects
